Warp
================================================================================

Warp to the another world


Requirements
--------------------------------------------------------------------------------

- [Peco](https://github.com/peco/peco)


Installation
--------------------------------------------------------------------------------

Download to your executable path.

    $ cd $HOME/bin
    $ curl -LO https://raw.github.com/Tomohiro/warp/master/p
    $ chmod +x ./warp


Usage
--------------------------------------------------------------------------------

```sh
$ warp
```
